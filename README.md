# Sistema de Asistencia - MVP

Sistema web para la toma de asistencia docente, desarrollado con:
- Frontend: HTML, CSS, JavaScript
- Backend: Supabase (Auth + Postgres + RLS)

## Funcionalidades
- Login de docentes
- Visualización de clases asignadas
- Toma de asistencia por grupo
- Persistencia segura con políticas RLS

## Estado
MVP funcional.
